-- Leggimi Cambia Tema --

Questo programma permette un rapido scambio di file x-server da fonti locali o remote.
I file remoti sono file validi provenienti da un host remoto nella cartella in cui ti trovi su un computer collegato. I file locali sono file presenti in sys o home.
ThemeChanger permette all'utente di selezionare una di queste due opzioni e di scaricare il contenuto su x-server.sys nella cartella sys, attivandolo senza necessita' di riavvio.
Il programma creera' anche una copia di tutti i file esistenti per preservare tutti gli stili per uso futuro.
